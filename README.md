# tic-tac-toe

### como correrlo

clonar el proyecto

- ```
  git clone git@github.com:gabrielurenah/tic-tac-toe.git
  ```

#### PvP
Tener tres terminales distintas abiertas
- ```
  python server.py
  ```
  
- ```
  python client.py
  ```
  
- ```
  python client.py
  ```

#### vs CPU
Abrir una terminal
- ```
  python3 vsMinimax.py
  ```
